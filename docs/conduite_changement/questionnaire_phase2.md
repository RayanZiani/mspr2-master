# Questionnaire — Phase 2 : Automatisation (Livrable 10)

## Objectif
Recueillir les besoins pour la phase 2 d'automatisation (actionneurs, décision automatique).

## Questions

1. Quelles actions manuelles souhaitez-vous automatiser en priorité ?
2. En cas de dérive de température, quel actionneur devrait se déclencher ? (ventilation, climatisation...)
3. Quel est le délai acceptable entre la détection d'une dérive et le déclenchement de l'actionneur ?
4. Qui doit valider l'action automatique avant exécution ? (supervision ou déclenchement direct ?)
5. Quels indicateurs de performance devrait afficher le tableau de bord de phase 2 ?
